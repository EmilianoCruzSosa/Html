Versiones de HTML

HTML 1.0

Ventajas:

  • Fue la primera versión de HTML que abrió el camino para la creación de páginas web.
  • Era fácil de aprender y usar.

Desventajas:

  • No tenía muchas características avanzadas.
  • Fue limitado en cuanto a la estructura y el diseño.

HTML 2.0

Ventajas:

  • Agregó nuevas etiquetas para mejorar la estructura y el diseño de las páginas web.
  • Permitió la inclusión de hojas de estilo CSS.

Desventajas:

  • Todavía era limitado en cuanto a la estructura y el diseño.

HTML 3.2

Ventajas:

  • Agregó muchas nuevas características para mejorar la estructura y el diseño de las páginas web.
  • Introdujo el modelo de caja de diseño para ayudar a definir el diseño de la página.

Desventajas:

  • Todavía había limitaciones en cuanto a la estructura y el diseño.

HTML 4.0

Ventajas:

  • Agregó muchas nuevas etiquetas y características para mejorar la estructura y el diseño de las páginas web.
  • Mejoró la accesibilidad de las páginas web.

Desventajas:

  • Todavía tenía algunas limitaciones en cuanto a la estructura y el diseño.

HTML 4.01

Ventajas:

  • Corrigió algunos errores y problemas de compatibilidad con navegadores.
  • Agregó algunas nuevas características para mejorar la estructura y el diseño de las páginas web.

Desventajas:

  • Aún tenía algunas limitaciones en cuanto a la estructura y el diseño.

XHTML 1.0

Ventajas:

  • Proporcionó una sintaxis más estricta y bien formada para HTML.
  • Introdujo nuevas etiquetas para mejorar la estructura
