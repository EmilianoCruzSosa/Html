Elementos basicos de HTML

Etiquetas

-Etiqueta "< strong >": para poner el texto en negrita.

-Etiqueta "< br >": para saltos de línea.

-Etiqueta "< img >": para añadir imágenes al documento.

-Etiqueta "< H1 >…< H6 >": para títulos dentro del contenido.

-Etiqueta "< ol >" para listas ordenadas, "< ul >" para listas desordenadas, "< li >" para elementos dentro de la lista.
